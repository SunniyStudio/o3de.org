**测试影响分析框架**（TIAF）是使用一种名为[快速测试影响分析](https://www.youtube.com/watch?v=mMzL1UCr0OE) 的技术来实现基于变更的测试，以优化自动审查的准备时间。TIAF 目前支持本地（C++）测试和 Python 测试。
