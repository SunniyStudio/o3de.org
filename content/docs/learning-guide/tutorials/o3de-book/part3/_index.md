# 第3部分 介绍组件通信
现在我们知道了如何创建组件，我们需要学习如何从多个组件构建逻辑。允许您做到这一点的下一个里程碑是某种形式的组件间通信。您需要各种组件与其他组件通信。

在 O3DE 中，组件有几种通信方式：
* 通过 FindComponent 实体方法直接访问。
* AZ::Interface - 类似单例的通信。
* AZ::Event - 通知机制。
* AZ::EBus - O3DE 中最通用、最强大的通信形式。
